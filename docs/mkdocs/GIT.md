# Github Integration


## Create GIT Repository

`git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/bissessr/iaac-docs.git
git push -u origin master`

## Push existing repository

`git remote add origin https://github.com/bissessr/iaac-docs.git
git push -u origin master`
